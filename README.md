# Smart Inventory & Sales Management System

A Python-based GUI application for managing warehouse stock and sales using a MySQL backend.

## Features
* **Product Management**: Add, update, and track stock levels.
* **Sales Tracking**: Record transactions and update inventory automatically.
* **Supplier & Purchase Records**: Keep track of vendors and incoming stock.
* **Database Driven**: Powered by MySQL for reliable data storage.

## Setup Instructions
1. Run the `gui_connect.sql` script in **MySQL Workbench** to set up the `inventory_db`.
2. Ensure you have the `mysql-connector-python` library installed.
3. Run `stock conect sql gui.py` to launch the application.
